## **SEO OPTIMIZATION CHECKLIST FOR EACH BLOG POST**

- [ ] Include target keyword in: Title, H1, first 100 words, natural placement throughout
- [ ] Include 2-3 related long-tail keywords
- [ ] Use descriptive H2/H3 subheadings with keywords
- [ ] Include meta description (150-160 chars) with target keyword
- [ ] Add internal links to service pages (#services, #pricing)
- [ ] Add internal links to other blog posts
- [ ] Include external links to authoritative healthcare sources
- [ ] Use FAQ schema for Q&A format posts
- [ ] Include alt text on all images describing the image
- [ ] Optimize first 100 words for keyword and readability
- [ ] Add CTA button linking to contact/booking page
- [ ] Include author bio with credentials
- [ ] Add publish date and "Last Updated" date
- [ ] Optimize URL slug: /blog/keyword-separated-by-hyphens
- [ ] Enable comments for engagement
